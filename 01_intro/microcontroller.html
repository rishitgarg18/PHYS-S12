### Week 4: Microcontroller Programming This week's assignment involved working on a Metro microcontroller and using it power a motor or an LED. Using the Arduino IDE application, we were able to load programs and send them directly to our Metro microcontroller, which powered our motor. This week's assignment required us to work on the following materials: Wires, Metro microntroller, breadboars, LEDs, Buttons,a multimeter, and resistors. In class, we learned we used the following 'Blink' program to make an LED light blink using a microcontroller.
